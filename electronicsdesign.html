Circuit

Using Ki-Cad, we got the battery, resistor, led, and button and put them onto the file.

We rearanged the components so that we could attatch wires to make everything connect correctly.

We linked each component to its real-life model by finding them on amazon, which allows us to retrive the products efficiently.

We put the system into the PCB editor to finalize the design..

We reprganized the wires to how we needed the circuit to be in the PCB editor.

We added the border, made the wires thicker, and make an engraving on the circuit.
